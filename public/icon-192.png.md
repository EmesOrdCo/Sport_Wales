# Favicon Generation Guide

This file describes the favicon assets needed. Use an online tool like:
- https://realfavicongenerator.net/
- https://favicon.io/

To generate PNG favicons from the SVG, or use the SVG directly in modern browsers.

## Required Sizes:
- 16x16 (favicon.ico)
- 32x32 (favicon.ico)
- 192x192 (Android)
- 512x512 (PWA)
